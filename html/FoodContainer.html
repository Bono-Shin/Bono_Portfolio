프로젝트 이름 FoodContainer
프로젝트 종류 팀 프로젝트
총 개발 기간 22.02.21 ~ 22.04.27
투입 인원 4
사용 된 기술 Java / Javascript / jQuery / CSS
HTML5 / JSP / Jackson / Ajax / Mysql
Spring Framework / JSTL / Bootstrap 5.1
프로젝트 요약 1인가구, 배달요금 인상, 요리의 번거로움 등 현대인들의
간편한 식사에 초점을 맞췄다.
또한, 단순히 주문만 해서 먹는 것이 아니라 서로 맛있게 먹었던
레시피 조합을 공유함으로써 '오늘 뭐먹지?' 라는 고민을 조금
덜어주기 위해 노력했다.
<맡은 파트>
→ 메인페이지, 상품 조회, 상품 상세조회, 최근 본 상품
비회원 장바구니 등록 시 쿠키사용, 회원의 상품 찜 등록
※ PC화면 - 가로 1024px, 모바일 - 아이폰SE에 최적화 되어 있습니다.
후기 반응형 웹 브라우저와 Spring Framework를 사용하는 것을
목표로 진행 했던 프로젝트였다.
Spring을 사용하는 것, 반응형 웹을 이해하는 것 크게 두 번의
고비가 있었지만, 잘 이겨낸 것 같다.
이번 작업으로 쿠키를 사용하는 법, Jackson 라이브러리를
이용한 비동기 통신, JSTL에 대해 많이 배울 수 있었고,
Spring Framework를 기본만큼이라도 사용할 수 있게 되어서
자신감이 조금 생긴 것 같다.
