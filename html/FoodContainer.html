총 개발 기간 22.02.21 ~ 22.04.27
투입 인원 4
사용 된 기술 Java / Javascript / jQuery / CSS
/ JSP / Json / Ajax / Mysql
프로젝트 요약 MVC1 까지 학습한 후 처음 진행 된 개인 프로젝트.
주제를 무엇으로 할까 고민하던 중, 평소에 자동차를 좋아해서
자동차에 관련 된 웹 페이지를 만들어 보기로 했다.
비록, 차량 수가 많지는 않지만 차량의 색상을 미리 입혀보거나,
옵션으로 제공 된 자동차 휠도 교체해 볼 수 있고,
트림 별 옵션 및 옵션에 대한 설명, 가격 등을 제공한다.
후기 첫 프로젝트 진행 후 가장 크게 달라진 점은,
그동안 배운 내용들을 가지고 생각한 모습대로 풀어낼 수
있어졌다는 점이다. (물론 굉장히 많이 헤맸다.)
전부는 아니더라도 생각한대로 구현한다는 것 또한 즐거웠다.
이전 직무와 비교했을 때, 아이템을 구현하는데 있어서는
제약이 없다고 느껴질 정도로 자유로운 것 같았다.
